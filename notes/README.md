# MD Watcher

一个轻量级的本地 Markdown 实时预览工具。

## 功能

- 📂 左侧目录树，快速浏览 Markdown 文件
- 🖥️ 右侧实时渲染预览
- 🔄 文件修改后自动刷新
- 💻 代码高亮
- 📊 表格、列表、引用等常用语法支持

## 快速开始

1. 启动服务：`start.bat` 或 `./start.sh`
2. 浏览器访问 `http://127.0.0.1:5000`
3. 点击左侧文件即可预览

## 示例

### 代码块

```python

    print("Hello, MD Watcher!")
```
